Instructions

Concentration, also known as Match Match, Memory, Pelmanism, Shinkei-suijaku, Pexeso or simply Pairs, is a card game in which all of the cards are laid face down on a surface and two cards are flipped face up over each turn. The object of the game is to turn over pairs of matching cards.

The Game

The reasoning behind someone wanting to play this game is that it has elements to it which utilize and excite the brain in many ways. Needing to use and work on your memory skills is a very important part of life and can even help against things like dimensia and alzheimers. There is also a competitive aspect to this game when playing with others, because of this it brings another aspect to the game where you need to try to find the most matches out of all of the other players in order to win, boosting your adrenaline and inner competitiveness.
